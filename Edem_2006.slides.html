Para una seccin rectangular se tiene $\Omega=\frac{1}{1+2 \kappa^{-1}(1+v)(\eta)^{2}}$

$\theta (x) = \theta_b (x) \cdot \varphi_b + \theta_s (x) \cdot \varphi_s$

Aqu, $\theta$ representa la rotacin total de la seccin transversal; $\theta_b$ es la rotacin por flexin y $\theta_s$ la rotacin promedio por cortante.

Adems, se debe cumplir con la condicin $\varphi_b + \varphi_s = 1$ (condicin de rotacin rgida).

Se propone lo siguiente $\theta(x)=\frac{d w (x)}{d x} \cdot \varphi_{b}+\theta_{s}(x) \cdot \varphi_{s}$

$M(x)= \underbrace{E I_1 \frac{\mathrm{d} ^2 w(x)}{\mathrm{d} x ^2} \varphi_b}_{M_1 (x)} + \underbrace{E I_2 \frac{\mathrm{d} \theta_s (x)}{\mathrm{d} x} \varphi_s}_{M_2 (x)}$

$b_1 = b \varphi_b$

$b_2 = b \varphi_s$

$\kappa_b = \frac{\mathrm{d} ^2 w(x)}{\mathrm{d} x ^2} $

$\kappa_s = \frac{\mathrm{d} \theta_s (x)}{\mathrm{d} x}$

Principio de la energía potencial mínima¶

$\Pi=\left(\kappa_{b}(w), \kappa_{s}\left(\theta_{s}\right)\right)=\Pi(w, \theta) \frac{1}{2} \int_{0}^{L}\left[E I_{1} \kappa_{b}^{2}\right] d x +\frac{1}{2} \int_{0}^{L}\left[E I_{2} \kappa_{s}^{2}\right] d x-\int_{0}^{L} q(x) w d x$

Se deriva el funcional

$d \Pi(w, \theta) [\delta w, \delta \theta_s] = \frac{1}{2} \int_{0}^{L}\left[\delta \kappa_b E I_{1} \kappa_b \right] d x +\frac{1}{2} \int_{0}^{L} \left[\delta \kappa_s E I_{2} \kappa_s \right] d x-\int_{0}^{L} q(x) \delta w d x = 0$

$\delta \kappa_b = \left [ \delta \left ( \frac{d^2 w}{d x^2} \right ) \right ]$

$\delta \kappa_s = \left [ \delta \left ( \frac{d \theta_s}{d x} \right ) \right ]$

De all se concluye que $w$ requiere continuidad $C_1$ y $\theta_s$ requiere $C_0$, lo cual implica que $w$ se debe interpolar con polinomios de Hermite, mientras que $\theta_s$ se interpola con polinomios de Lagrange.

# Programa para deducir la matriz de rigidez de un elemento de viga de
# Timoshenko a partir de la solucion de la ecuacion diferencial

# %%Importacin de libreras
from sympy import symbols, integrate, pretty, simplify, zeros, solve, limit
from sympy import Matrix

# %%Defincin de variables
x, L, V, M, t, w, EI, GAast = symbols('x L  V M t w EI GAast')
C1, C2, C3, C4 = symbols('C1, C2, C3, C4')  # Constantes de integracin

q = 0

# %%Se calcula la matriz de rigidez
K_T = zeros(4)

V = integrate(q, x) + C1  # Ecuacines calculadas con constantes.
M = integrate(V, x) + C2
t = (integrate(M, x) + C3)/EI
w = integrate(t - V/GAast, x) + C4
